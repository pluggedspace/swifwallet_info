    Introduction

    Swif is a modern cross-border international wallet designed to facilitate easy, cost-effective, and secure financial transactions. It leverages the Stellar blockchain network for seamless USDC transactions, with integrations for ramps, investing, and utility payments. Swif aims to redefine financial inclusion for its users by focusing on low fees, user-friendliness, and scalability.

    Features

    Core Features

    • Cross-Border Remittance: Instant transactions across borders with minimal fees, powered by Stellar blockchainâ€™s USDCXLM.
    • Multi-Currency Support: Store and transact in multiple currencies with on-the-fly conversion to USDC.
    • Vendor Integration: Prefill vendor details based on user profiles and country-specific availability.
    • Security: Encryption standards for sensitive data and two-factor authentication for account protection.
    • User-Centric Design: Easy-to-use interface powered by Flutter and pre-fetching of user details for seamless onboarding.

    Planned Features

    • IBAN Functionality: Allow users to create virtual IBANs for banking activities.
    • DeFi Integration: Stellar-based DeFi protocols for staking, liquidity pools, and token swaps.
    • Investments & Utilities: Options for investing in assets and paying bills directly from the wallet.
    • Commerce & Pooled Shopping: Group shopping features for shared purchases.

    System Design

    Architecture Overview

    • Frontend: Built with Flutter for cross-platform support.
    • Backend: Written in Python with Django, integrates Stellar SDK for blockchain functionality.
    • Database: PostgreSQL.
    • Hosting: Deployed on AWS Free Tier for MVP.

    Transaction Workflow

    1. User initiates transaction by selecting vendor and payment method.
    2. Transaction processing: Fees calculated dynamically and converted to USDCXLM for blockchain execution.
    3. Settlement: Instant credit to recipient's account or vendor.
