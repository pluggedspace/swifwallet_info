Introduction

Swif is a modern cross-border international wallet designed to facilitate easy, cost-effective, and secure financial transactions. It leverages the Stellar blockchain network for seamless USDC transactions, with integrations for ramps, investing, and utility payments. Swif aims to redefine financial inclusion for its users by focusing on low fees, user-friendliness, and scalability.

  • User-Centric Design: Easy-to-use interface powered by Flutter and pre-fetching of user details for seamless onboarding.

System Design

Architecture Overview

  • Frontend: Built with Flutter for cross-platform support.
  • Backend: Written in Python with Django, integrates Stellar SDK for blockchain functionality.
  • Database: PostgreSQL.
  • Hosting: Deployed on AWS Free Tier for MVP.
